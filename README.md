# Export Album Data from MusicBrainz

## Installation

```
mkdir musicbrainz-jsondump
virtualenv -p /usr/bin/python3.5 musicbrainz-jsondump-env
cd musicbrainz-jsondump
pip install notebook SQLAlchemy psycopg2 PyLyrics
```
